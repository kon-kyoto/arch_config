## ⌨️ Горячие клавиши Kitty

### Размер шрифта
| Комбинация | Действие |
|------------|----------|
| `Ctrl + Plus` | Увеличить шрифт (+0.5) |
| `Ctrl + Minus` | Уменьшить шрифт (–0.5) |
| `Ctrl + 0` | Сбросить до 13.5 |

### Буфер обмена
| Комбинация | Действие |
|------------|----------|
| `Ctrl + Shift + C` | Копировать |
| `Ctrl + Shift + V` | Вставить |

### Управление окнами и вкладками
| Комбинация | Действие |
|------------|----------|
| `Ctrl + Shift + Q` | Закрыть окно |
| `Ctrl + Shift + T` | Создать вкладку |
| `Ctrl + Shift + Enter` | Новое окно |
| `Ctrl + Tab` | Следующая вкладка |
| `Ctrl + Shift + Tab` | Предыдущая вкладка |

### Прокрутка
| Комбинация | Действие |
|------------|----------|
| `Ctrl + Shift + Up` | Прокрутить вверх |
| `Ctrl + Shift + Down` | Прокрутить вниз |

## 💡 Полезные мелочи

- **Повышенная контрастность**: красные цвета насыщеннее для лучшей читаемости
- **Затемнение неактивных окон**: `dim_opacity 0.4` помогает фокусироваться на активном терминале
- **Плавное изменение шрифта**: шаг `+0.5` вместо стандартного
- **Иконки Nerd Font**: добавлена карта символов для иконок
