# 🖥️ Hyprland — краткая документация

## 🚀 Запускаемые программы (autostart)

| Программа | Назначение |
|-----------|------------|
| `waybar` | Верхняя панель (статус-бар) |
| `awww-daemon` | Анимации обоев / демон |

> 📌 `exec-once` в `hyprland.conf`

---

## ⌨️ Горячие клавиши (Hotkeys)

| Комбинация | Действие |
|------------|----------|
| `SUPER + Q` | Открыть терминал (Kitty) |
| `SUPER + C` | Закрыть активное окно |
| `SUPER + E` | Открыть файловый менеджер (Thunar) |
| `SUPER + B` | Открыть браузер (Firefox) |
| `SUPER + V` | Переключить плавающий режим окна |
| `SUPER + R` | Открыть меню запуска приложений (Wofi) |
| `SUPER + P` | Псевдотайлинг (Dwindle) |
| `SUPER + U` | Разделить окно (togglesplit) |
| `SUPER + G` | Заблокировать экран (hyprlock) |
| `SUPER + S` | Открыть/закрыть спец. workspace (scratchpad) |
| `SUPER + SHIFT + S` | Переместить окно в scratchpad |
| `SUPER + H/L/K/J` | Переместить фокус (влево/вправо/вверх/вниз) |
| `SUPER + [1-9,0]` | Переключиться на workspace 1–10 |
| `SUPER + SHIFT + [1-9,0]` | Переместить окно на workspace |
| `SUPER + SHIFT + R` | Перезагрузить Hyprland (reload) |
| `SUPER + SHIFT + E` | Выйти из Hyprland |
| `SUPER + мышь (ЛКМ)` | Переместить окно |
| `SUPER + мышь (ПКМ)` | Изменить размер окна |
| `PRINT` | Скриншот области (hyprshot) |

### 🔊 Медиа-клавиши

| Клавиша | Действие |
|---------|----------|
| `XF86AudioRaiseVolume` | Громкость +5% |
| `XF86AudioLowerVolume` | Громкость –5% |
| `XF86AudioMute` | Выкл/вкл звук |
| `XF86AudioMicMute` | Выкл/вкл микрофон |
| `XF86AudioNext` | Следующий трек |
| `XF86AudioPrev` | Предыдущий трек |
| `XF86AudioPlay/Pause` | Play/Pause |
| `F4` | Яркость –5% |
| `F5` | Яркость +5% |

## 🧩 Workspaces

- **Обычные**: `1` – `10`
- **Специальный (scratchpad)**: `magic` (клавиша `SUPER + S`)

---

## 🎨 Оформление

| Параметр | Значение |
|----------|----------|
| Зазоры | `in: 5`, `out: 20` |
| Скругление | `10` |
| Активная рамка | градиент `#33ccff → #00ff99` |
| Неактивная рамка | `#595959` |
| Тени | включены |
| Размытие | включено (size=3, passes=1) |
| Анимации | включены |

---

## ⌨️ Раскладка клавиатуры

- **Раскладки**: `us, ru`
- **Переключение**: `Alt + Shift` (обе стороны)
- **Мониторинг**: скрипт `keyboard-monitor.sh` обновляет `/tmp/keyboard_layout` и шлёт сигнал Waybar (`RTMIN+8`)

> В `hyprlock.conf` можно отобразить текущую раскладку через `$LAYOUT`

---

## 🔒 Блокировка экрана (hyprlock)

| Элемент | Описание |
|---------|----------|
| Фон | `~/.config/wallpapers/lain.png` + размытие |
| Приветствие | «Тебе тут не рады...» |
| Часы | шрифт DSEG7 Classic, размер 100 |
| Поле ввода пароля | с индикацией ошибки |
| Раскладка | отображается снизу (`$LAYOUT`) |

## 📁 Полезные файлы

| Путь | Назначение |
|------|------------|
| `~/.config/hypr/hyprland.conf` | Основной конфиг Hyprland |
| `~/.config/hypr/hyprlock.conf` | Конфиг блокировки экрана |
| `~/.config/hypr/scripts/keyboard-monitor.sh` | Мониторинг раскладки |
| `/tmp/keyboard_layout` | Кэш текущей раскладки |
| `~/.config/waybar/scripts/keyboard-layout.sh` | Скрипт получения раскладки |
